# Lumina Learning - Speed Optimization Task

This is a simple static landing page project for website speed optimization practice.

## Files included
- index.html
- styles.css
- script.js
- images folder with SVG assets

## Required work
Please optimize only this page without changing the design.

Tasks:
1. Add lazy loading to images where suitable.
2. Minify CSS and JS files.
3. Optimize image loading.
4. Suggest or add Gzip Compression settings if server access is available.
5. Suggest or add Browser Caching settings if server access is available.
6. Send a short report explaining what was changed.

## Expected delivery
- Updated optimized files.
- Short note explaining the changes.
- Screenshot before and after from PageSpeed Insights, GTmetrix, or Lighthouse.
